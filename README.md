




## Installation

- 1- nodejs

- 2- postgressql


## .env
then you should edit the databaseurl inside .env file


## Running the server
after that run the following in cmd/terminal:

```bash
1- npm install
2- npm i --save-dev prisma@latest 
3- npm run reset
4- npm start
```
